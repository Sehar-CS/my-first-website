# Amazon Homepage Clone

A frontend web development project that replicates the homepage of Amazon. The main goal of this project was to practice modern web layout techniques and create a clean, structured, and visually accurate user interface using pure HTML and CSS without any external frameworks.

## 🚀 Features
* **Navigation Bar:** Recreated the detailed Amazon top navigation bar with a search container, delivery location icon, and account dropdown layouts.
* **Product Grids:** Used modern CSS techniques to design the multi-card product layout grid seen on the actual Amazon homepage.
* **Hero Section:** Implemented a large image slider/banner layout background with overlay content cards.
* **Responsive Layout:** Designed to be flexible across different screen sizes.
* **Hover Effects:** Smooth transitions on buttons, links, and product cards for a better user experience.

## 🛠️ Technologies Used
* **HTML5:** For semantic and structured web layout.
* **Vanilla CSS:** Custom styling, layout design (using Flexbox and Grid), and interactive hover elements.
